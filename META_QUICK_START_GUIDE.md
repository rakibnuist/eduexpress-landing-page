## ğŸ”§ Quick Troubleshooting

### "My ad was rejected"
**Common reasons:**
- Too much text in image (>20%)
- Misleading claims
- Poor landing page experience

**Fix:** Reduce image text, ensure claims are accurate, optimize page speed

### "High cost per click but no conversions"
**Problem:** Landing page issue

**Fix:** 
- Check mobile experience
- Ensure form is working
- Verify Pixel is firing CompleteRegistration event

### "Low CTR (<1%)"
**Problem:** Ad creative not engaging

**Fix:**
- Use more emotional/urgent copy
- Test different images
- Try video instead of static image

### "Getting leads but all are spam/fake"
**Problem:** Targeting too broad or form too easy

**Fix:**
- Narrow age range (e.g., 19-24 instead of 18-25)
- Add phone number validation
- Include GPA/qualification question
